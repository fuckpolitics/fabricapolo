# Редактирование фотографий товаров

## Обзор системы

Система редактирования фотографий предназначена **только для разработчика** и позволяет изменять изображения товаров в каталоге с **постоянным сохранением** в код проекта.

## Как работает сохранение

Система использует двухэтапный процесс сохранения:

1. **Временное сохранение** - изменения применяются немедленно в интерфейсе
2. **Постоянное сохранение** - через экспорт обновленного кода в файл проекта

## Как включить режим редактирования

### Для разработчика:

1. Откройте файл `src/components/CategoryPage.vue`
2. Найдите в секции `data()` параметр `editMode`
3. Измените значение `editMode: false` на `editMode: true`
4. Сохраните файл

```javascript
data() {
  return {
    products: [],
    // ФЛАГ ДЛЯ РАЗРАБОТЧИКА: включить режим редактирования фотографий
    // Измените на false перед выгрузкой на продакшн
    editMode: true,  // <-- ИЗМЕНИТЕ ЗДЕСЬ
    editingProductId: null
  }
},
```

## Пошаговая инструкция для разработчика

### Шаг 1: Включение режима редактирования

1. В файле `CategoryPage.vue` установите `editMode: true`
2. Обновите страницу в браузере
3. Появится **Панель разработчика** с кнопками управления

### Шаг 2: Редактирование фотографий

1. Нажмите кнопку **"Управление фото"** на нужном товаре
2. Откроется панель ImageManager
3. Внесите изменения:
   - Добавьте новые изображения (URL или загрузка файлов)
   - Удалите ненужные изображения
   - Измените порядок изображений

### Шаг 3: Сохранение изменений

После каждого изменения автоматически:
- Изображения обновляются в интерфейсе
- Данные сохраняются в памяти
- Появляется уведомление о необходимости экспорта кода

### Шаг 4: Экспорт обновленного кода

**Способ 1: Экспорт в файл**
1. Нажмите кнопку **"📄 Экспорт кода"** в Панели разработчика
2. Скачается файл `category_[имя_категории]_updated.js`
3. Откройте этот файл и скопируйте содержимое

**Способ 2: Копирование в буфер обмена**
1. Нажмите кнопку **"📋 Копировать код"** в Панели разработчика
2. Код автоматически скопируется в буфер обмена

### Шаг 5: Обновление кода проекта

1. Откройте файл `src/components/CategoryPage.vue`
2. Найдите секцию `loadProducts()` метод
3. Найдите соответствующую категорию (polo, uniform, hoodies, shoppers, jackets)
4. **Полностью замените** массив товаров на новый код
5. Сохраните файл

### Шаг 6: Проверка и выгрузка

1. Обновите страницу - изменения должны быть видны
2. Проверьте все изображения и их порядок
3. Установите `editMode: false` перед выгрузкой на продакшн

## Важные особенности

- **Постоянное сохранение**: Изменения сохраняются в коде проекта
- **Безопасность**: Пользователи не видят инструменты редактирования
- **Контроль версий**: Все изменения отслеживаются через Git
- **Резервное копирование**: Старый код сохраняется в истории Git

## Техническая реализация

### Компоненты

- **CategoryPage.vue** - основная страница с панелью разработчика
- **ProductGallery.vue** - галерея изображений товара
- **ImageManager.vue** - панель управления фотографиями
- **imageManager.js** - утилиты для управления изображениями

### Функции сохранения

- `saveChanges()` - сохраняет изменения и генерирует код
- `exportCode()` - экспортирует код в файл
- `copyCodeToClipboard()` - копирует код в буфер обмена
- `generateUpdatedCategoryCode()` - генерирует обновленный код

## Процесс работы с фотографиями

### 1. Подготовка
```javascript
// Включаем режим для разработки
editMode: true
```

### 2. Редактирование
- Откройте сайт локально
- Используйте Панель разработчика и ImageManager
- Вносите изменения в фотографии

### 3. Сохранение
- Нажмите "Экспорт кода" или "Копировать код"
- Замените соответствующий раздел в CategoryPage.vue
- Проверьте результат

### 4. Выгрузка на продакшн
```javascript
// Выключаем режим для продакшн
editMode: false
```

## Советы по использованию

1. **Регулярно экспортируйте код** после значительных изменений
2. **Делайте коммиты в Git** перед крупными изменениями
3. **Проверяйте URL изображений** - они должны быть доступны
4. **Тестируйте на разных устройствах** после обновления
5. **Сохраняйте резервные копии** старого кода

## Поиск и устранение проблем

### Кнопки управления фото не видны
- Убедитесь, что `editMode: true` в коде
- Проверьте, что вы перезагрузили страницу

### Экспортированный код не работает
- Проверьте, что вы заменили весь массив категории
- Убедитесь в правильности синтаксиса JavaScript
- Проверьте наличие закрывающих скобок и запятых

### Изображения не отображаются после обновления
- Проверьте URL изображений на доступность
- Убедитесь, что изображения загружаются в браузере
- Проверьте консоль браузера на ошибки

### Панель разработчика не видна
- Убедитесь, что `editMode: true`
- Проверьте консоль на ошибки JavaScript

## ВАЖНО: Проверка перед выгрузкой

Перед выгрузкой на продакшн обязательно:

1. ✅ Установите `editMode: false`
2. ✅ Проверьте, что Панель разработчика не видна
3. ✅ Протестируйте все категории с измененными изображениями
4. ✅ Проверьте загрузку изображений на разных устройствах
5. ✅ Сделайте коммит в Git с описанием изменений

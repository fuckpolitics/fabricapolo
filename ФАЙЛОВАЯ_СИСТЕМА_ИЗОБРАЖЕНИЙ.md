# Файловая система изображений

## Обзор системы

Система автоматически сканирует директории с изображениями и отображает их в слайдерах товаров. Никаких элементов редактирования на сайте - просто добавляйте файлы в папки!

## Как работает система

### 1. Автоматическое сканирование
- Система сканирует директорию каждого товара
- Находит все файлы изображений (jpg, jpeg, png, gif, webp, bmp)
- Автоматически добавляет их в слайдер

### 2. Умная сортировка
Система сортирует изображения по правилам:

1. **Числовые префиксы** в начале имени файла:
   - `1.jpg`, `2.png`, `3.jpeg` → будут в порядке 1, 2, 3
   - `10.jpg`, `2.jpg` → будут в порядке 2, 10 (правильная числовая сортировка)

2. **Остальные файлы** в алфавитном порядке:
   - `detail.jpg`, `front.jpg`, `special.png` → detail, front, special

3. **Смешанная сортировка**:
   - Файлы с числовыми префиксами идут первыми
   - Потом остальные файлы в алфавитном порядке

## Примеры сортировки

### Пример 1: Полная числовая сортировка
```
Файлы: 3.jpg, 1.jpg, 2.jpg, 10.jpg
Результат: 1.jpg, 2.jpg, 3.jpg, 10.jpg
```

### Пример 2: Смешанная сортировка
```
Файлы: detail.jpg, 2.jpg, front.jpg, 1.jpg, special.png
Результат: 1.jpg, 2.jpg, detail.jpg, front.jpg, special.png
```

### Пример 3: Только первые три упорядочены
```
Файлы: 3.jpg, 1.jpg, 2.jpg, detail.jpg, special.jpg, fabric.png
Результат: 1.jpg, 2.jpg, 3.jpg, detail.jpg, fabric.jpg, special.png
```

### Пример 4: Без числовых префиксов
```
Файлы: back.jpg, detail.jpg, front.jpg, side.jpg
Результат: back.jpg, detail.jpg, front.jpg, side.jpg
```

## Поддерживаемые форматы

Система автоматически распознает изображения следующих форматов:

- **JPEG/JPG** - `.jpg`, `.jpeg`
- **PNG** - `.png`
- **GIF** - `.gif`
- **WebP** - `.webp`
- **BMP** - `.bmp`

## Как добавить изображения

### Шаг 1: Найдите нужную директорию
Определите категорию и slug (английское имя папки) товара:
- Категория: `polo`, `uniform`, `hoodies`, `shoppers`, `jackets`
- Slug товара: поле `slug` в `products.json` или в `CategoryPage.vue` (например: `female-polo`, `unarmy`, `vests`)

### Шаг 2: Добавьте файлы
Скопируйте изображения в соответствующую папку:
```
src/img/products/[категория]/[slug]/
```

### Шаг 3: Проверьте результат
Обновите страницу - изображения автоматически появятся в слайдере!

## Сценарии использования

### Сценарий 1: Полный порядок
Хотите строгий порядок всех изображений:
```
1.jpg (главное фото)
2.jpg (вид сбоку)
3.jpg (вид сзади)
detail-1.jpg (деталь 1)
detail-2.jpg (деталь 2)
```

### Сценарий 2: Только важные фото упорядочены
Важные фото в начале, остальные в любом порядке:
```
1.jpg (главное фото)
2.jpg (важный ракурс)
3.jpg (деталь)
back.jpg (задний вид)
side.jpg (боковой вид)
texture.jpg (текстура)
```

### Сценарий 3: Без упорядочивания
Все фото в алфавитном порядке:
```
back.jpg
detail.jpg
front.jpg
side.jpg
```

## Техническая реализация

### Компоненты
- `CategoryPage.vue` - основной компонент
- `ProductGallery.vue` - галерея изображений
- `fileSystemManager.js` - утилита для работы с файлами

### Методы
- `getImagesFromDirectory(category, productSlug)` - сканирование директории по категории и slug товара
- `sortImageFiles()` - умная сортировка
- `extractNumericPrefix()` - извлечение числовых префиксов

## Поиск и устранение проблем

### Изображения не отображаются
1. Проверьте путь к директории
2. Убедитесь, что файлы имеют правильные расширения
3. Проверьте права доступа к файлам

### Неправильный порядок
1. Проверьте числовые префиксы в именах файлов
2. Убедитесь, что числа в начале имени файла
3. Проверьте, что нет пробелов перед числами

### Файлы не распознаются как изображения
1. Проверьте расширения файлов (.jpg, .png, .jpeg и т.д.)
2. Убедитесь, что файлы не повреждены
3. Проверьте регистр расширений (должны быть в нижнем регистре)

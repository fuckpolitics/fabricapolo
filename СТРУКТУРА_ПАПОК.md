# Структура папок для изображений товаров

## Созданные директории

```
src/img/products/
├── polo/                          # Футболки поло
│   ├── female-polo/               # Футболка поло женская
│   ├── male-polo/                 # Футболка поло мужская
│   ├── polo-details/              # Футболка поло детали
│   └── basic-tshirt/              # Футболка
├── uniform/                       # Форменная одежда
│   ├── unarmy/                    # Юнармия
│   ├── unarmy-hoodies/            # Юнармия. Толстовки
│   └── ems/                       # МЧС
├── hoodies/                       # Трикотаж
│   ├── sweatshirts/               # Толстовки
│   ├── hoodie/                    # Худи
│   ├── sweaters/                  # Свитшоты
│   ├── bomber/                    # Бомберы
│   └── pants/                     # Брюки
├── shoppers/                      # Промо текстиль
│   └── promo-textile/             # Промо текстиль
└── jackets/                       # Жилетки/куртки/ветровки
    ├── vests/                     # Жилетки
    ├── coats/                     # Куртки
    └── windbreaker/               # Ветровки
```

Папки названы английскими словами (slug) для удобства навигации и понимания.

## Как использовать

### 1. Добавление изображений
Просто скопируйте файлы изображений в соответствующую папку:

```bash
# Пример для футболки поло женской
cp /path/to/image1.jpg src/img/products/polo/female-polo/1.jpg
cp /path/to/image2.png src/img/products/polo/female-polo/2.png
cp /path/to/detail.jpg src/img/products/polo/female-polo/detail.jpg
```

### 2. Правила именования файлов

#### Для строгого порядка:
```
1.jpg     # Главное фото
2.png     # Вид сбоку
3.jpeg    # Вид сзади
```

#### Для смешанного порядка:
```
1.jpg     # Главное фото
2.jpg     # Важный ракурс
detail.jpg # Детали
back.jpg  # Задний вид
```

#### Без упорядочивания:
```
front.jpg
side.jpg
back.jpg
detail.jpg
```

### 3. Поддерживаемые форматы
- **JPEG/JPG** - `.jpg`, `.jpeg`
- **PNG** - `.png`
- **GIF** - `.gif`
- **WebP** - `.webp`
- **BMP** - `.bmp`

## Проверка работы

1. Добавьте изображения в любую папку
2. Обновите страницу соответствующей категории
3. Изображения автоматически появятся в слайдере

## Система сортировки

Система автоматически сортирует файлы по правилам:

1. **Сначала файлы с числовыми префиксами** в числовом порядке
2. **Потом остальные файлы** в алфавитном порядке

### Примеры сортировки:
```
Входные файлы: 3.jpg, 1.jpg, detail.jpg, 2.jpg, front.jpg
Результат: 1.jpg, 2.jpg, 3.jpg, detail.jpg, front.jpg
```

```
Входные файлы: back.jpg, detail.jpg, front.jpg, side.jpg
Результат: back.jpg, detail.jpg, front.jpg, side.jpg
```

## Резервное копирование

Для резервного копирования изображений:
```bash
# Создать архив всех изображений
tar -czf products_images_backup.tar.gz src/img/products/

# Восстановить из архива
tar -xzf products_images_backup.tar.gz
```

## Автоматизация

Для массового добавления изображений можно использовать скрипты:

```bash
#!/bin/bash
# Пример скрипта для добавления изображений

PRODUCT_DIR="src/img/products/polo/female-polo"
IMAGES_DIR="/path/to/source/images"

# Копировать все изображения
cp "$IMAGES_DIR"/* "$PRODUCT_DIR/"

# Переименовать для порядка
count=1
for file in "$PRODUCT_DIR"/*.jpg; do
  mv "$file" "$PRODUCT_DIR/$count.jpg"
  ((count++))
done
```
